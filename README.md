# karaoke-style-lyrics
This repository provides the highest quality **karaoke-style lyrics** in `.ass` format.

## Lyrics Sources (in order of priority):

1. **Hardcoded Subtitles** from the Official YouTube MV
2. **Closed Captions** from the Official YouTube MV
3. **Description** of the Official YouTube MV
4. **Musixmatch**
5. **QQ Music**
6. **NetEase Cloud Music**

## File Naming Convention:

`artist/album/discnumber-track-title.ass`
